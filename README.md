## 📌 Project Overview

This project analyzes the Disney+ Hotstar content dataset using Microsoft Excel and Power BI.

The dataset was downloaded from Kaggle, cleaned in Microsoft Excel, and transformed into an interactive Power BI dashboard that provides insights into content distribution, genres, runtime, age ratings, release trends, and content types.

---

```
Kaggle Dataset
      │
      ▼
Data Cleaning (Microsoft Excel)
      │
      ▼

      │
      ▼
Power BI
      │

Interactive Dashboard
```

## 📂 Dataset

**Source:** Kaggle

The dataset contains information about Disney+ Hotstar content, including:

- Title
- Genre
- Release Year
- Runtime
- Age Rating
- Content Type (Movie / TV Show)

---

## 🧹 Data Cleaning

The dataset was cleaned using Microsoft Excel.

Cleaning steps included:

- Removed duplicate records
- Handled missing values
- Corrected inconsistent values
- Standardized text formatting
- Verified data types
- Prepared the dataset for visualization

## 📊 Business Questions Answered

- Which genres have the highest number of titles?
- What is the ratio of Movies to TV Shows?

- How has content production changed over the years?
- How has average runtime changed over time?
- Which genres have the highest total runtime?

---

## 📌 Key Insights

- The platform contains **6,670** titles.
- There are **40** different genres.
- Movies make up the majority of the content library.
- Most content falls under a few popular age ratings.
- Runtime varies significantly across genres.
- Content releases increased rapidly in recent years before declining.
